### Regras:

## Objetivo
Afundar todos os navios do adversário antes que ele afunde os teus.

## Preparação
- Cada jogador tem uma grelha (ex: 10x10).
- Cada jogador posiciona secretamente os seus navios na grelha.
- Os navios podem ser colocados na horizontal ou vertical.
- Os navios não podem sobrepor-se.

### Exemplo de frota (padrão)
- 1 Porta-aviões (5 casas)
- 1 Couraçado (4 casas)
- 2 Cruzadores (3 casas)
- 1 Submarino (3 casas)
- 1 Destroyer (2 casas)

*(Podes ajustar a frota conforme o teu projeto.)*

## Como jogar
1. Os jogadores jogam à vez.
2. Na sua vez, o jogador escolhe uma coordenada (ex: B5) para atacar.
3. O adversário responde:
   - **Água** se não houver navio
   - **Acertou** se atingiu parte de um navio
   - **Afundou** se destruiu completamente um navio
4. O jogador marca o resultado na sua grelha de tiros.

## Fim do jogo
O jogo termina quando todos os navios de um jogador forem afundados.  
O outro jogador é o vencedor.
